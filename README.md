## System Configuration

To run the app:

* Ruby version: 2.2.3 

* System dependencies: 

``` bash
$ bundle install
```

* Environment variables: 

FOOD2FORK_KEY = <Key to https://food2fork.com/about/api>
FOOD2FORK_SERVER_AND_PORT = <the server and port> ('www.food2fork.com:80' by default)
